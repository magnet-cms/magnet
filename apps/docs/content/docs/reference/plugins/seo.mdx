# SEO Plugin Reference

The `@magnet-cms/plugin-seo` package is currently in development.

## Status

The plugin package is published as a stub. It does not yet export a `forRoot()` provider. When it ships, registration will follow the standard plugin pattern.

## Planned Features

- Meta tags management (title, description, keywords)
- Open Graph tags for social sharing
- Twitter Cards metadata
- Structured data (JSON-LD schema.org)
- Sitemap generation
- Robots.txt management
- Canonical URL handling
