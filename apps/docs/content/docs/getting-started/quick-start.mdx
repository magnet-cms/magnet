# Quick Start

Follow this guide to set up a basic Magnet application.

## Installation

First, install the required packages:

```bash
bun add @magnet/core @magnet/adapter-mongoose @magnet/common
```

## Basic Setup

Create your main application module:

```typescript
import { Module } from '@nestjs/common'
import { MagnetModule } from '@magnet/core'
import { MongooseAdapter } from '@magnet/adapter-mongoose'

@Module({
  imports: [
    MagnetModule.forRoot({
      adapter: new MongooseAdapter({
        uri: 'mongodb://localhost:27017/magnet',
      }),
    }),
  ],
})
export class AppModule {}
```

## Running the Application

Start your application:

```bash
bun run start:dev
```

Your Magnet application is now running! The admin UI is available at `http://localhost:3000/admin`.
