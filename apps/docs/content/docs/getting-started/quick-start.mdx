# Quick Start

Follow this guide to set up a basic Magnet application.

## Installation

First, install the required packages:

```bash
bun add @magnet-cms/core @magnet-cms/adapter-db-mongoose @magnet-cms/common
```

## Basic Setup

Create your main application module:

```typescript
import { Module } from '@nestjs/common'
import { MagnetModule } from '@magnet-cms/core'
import { MongooseDatabaseAdapter } from '@magnet-cms/adapter-db-mongoose'

@Module({
  imports: [
    MagnetModule.forRoot([
      MongooseDatabaseAdapter.forRoot(),
    ], { admin: true }),
  ],
})
export class AppModule {}
```

Environment variables like `MONGODB_URI` and `JWT_SECRET` are auto-resolved from `process.env` — no need to pass them explicitly.

## Running the Application

Start your application:

```bash
bun run start:dev
```

Your Magnet application is now running! The admin UI is available at `http://localhost:3000/admin`.
