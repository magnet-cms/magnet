# Getting Started

This section covers everything you need to know to get started with Magnet.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18 or higher
- Bun (recommended) or npm/yarn
- MongoDB (or your preferred database)

## Quick Start

1. Install Magnet packages:

```bash
bun add @magnet/core @magnet/adapter-mongoose
```

2. Configure your NestJS application to use Magnet.

Check out the [Quick Start Guide](/docs/getting-started/quick-start) for a complete walkthrough.
