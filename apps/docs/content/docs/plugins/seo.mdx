The SEO plugin adds search engine optimization capabilities to your Magnet CMS content.

## Status

The SEO plugin is currently in development. This documentation will be updated as features are implemented.

## Planned Features

- **Meta Tags Management**: Title, description, keywords
- **Open Graph Tags**: Social media sharing metadata
- **Twitter Cards**: Twitter-specific metadata
- **Structured Data**: JSON-LD schema.org markup
- **Sitemap Generation**: Automatic XML sitemap
- **Robots.txt Management**: Crawler directives
- **Canonical URLs**: Duplicate content handling

## Installation (Coming Soon)

```bash
bun add @magnet-cms/plugin-seo
```

## Configuration (Coming Soon)

The `@magnet-cms/plugin-seo` package does not yet export a `forRoot()` provider. When it ships, registration will follow the same pattern as other plugins: add the plugin’s provider to the **first** argument of `MagnetModule.forRoot` (the provider array), alongside your database adapter.

```typescript
import { Module } from '@nestjs/common'
import { MagnetModule } from '@magnet-cms/core'
import { MongooseDatabaseAdapter } from '@magnet-cms/adapter-db-mongoose'
// import { SeoPlugin } from '@magnet-cms/plugin-seo'

@Module({
  imports: [
    MagnetModule.forRoot([
      MongooseDatabaseAdapter.forRoot(),
      // SeoPlugin.forRoot(),
    ]),
  ],
})
export class AppModule {}
```
