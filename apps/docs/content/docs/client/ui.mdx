The `@magnet-cms/ui` package provides a comprehensive set of UI components based on shadcn/ui, styled with Tailwind CSS.

## Installation

```bash
bun add @magnet-cms/ui
```

## Setup

### Tailwind Configuration

Add the UI package to your Tailwind content paths:

```typescript
// tailwind.config.ts
import type { Config } from 'tailwindcss'

const config: Config = {
  darkMode: 'class',
  content: [
    './src/**/*.{ts,tsx}',
    './node_modules/@magnet-cms/ui/dist/**/*.js',
  ],
}

export default config
```

### Import Styles

Import the base styles in your app:

```typescript
import '@magnet-cms/ui/styles.css'
```

## Utilities

### cn() - Class Name Merge

Utility for merging Tailwind classes:

```tsx
import { cn } from '@magnet-cms/ui/lib/utils'

<div className={cn('base-class', isActive && 'active-class', className)} />
```

## Dark Mode

Components automatically support dark mode via the `dark` class:

```tsx
// In your root layout
<html className="dark">
  {/* ... */}
</html>
```

Or use a theme provider:

```tsx
import { ThemeProvider } from '@magnet-cms/ui/providers'

<ThemeProvider defaultTheme="system">
  <App />
</ThemeProvider>
```

## Customization

Override component styles with Tailwind classes:

```tsx
<Button className="bg-purple-600 hover:bg-purple-700">
  Custom Button
</Button>
```

Or create custom variants:

```tsx
import { Button } from '@magnet-cms/ui/components'
import { cn } from '@magnet-cms/ui/lib/utils'

function PurpleButton({ className, ...props }) {
  return (
    <Button
      className={cn('bg-purple-600 hover:bg-purple-700', className)}
      {...props}
    />
  )
}
```
