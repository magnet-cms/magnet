# Resend email adapter

[`@magnet-cms/email-resend`](https://www.npmjs.com/package/@magnet-cms/email-resend) implements `EmailMagnetProvider` for `MagnetModule.forRoot([...])`.

## Installation

```bash
bun add @magnet-cms/email-resend resend
```

## Registration

```typescript
import { Module } from '@nestjs/common'
import { MagnetModule } from '@magnet-cms/core'
import { MongooseDatabaseAdapter } from '@magnet-cms/adapter-db-mongoose'
import { ResendEmailAdapter } from '@magnet-cms/email-resend'

@Module({
  imports: [
    MagnetModule.forRoot(
      [
        MongooseDatabaseAdapter.forRoot(),
        ResendEmailAdapter.forRoot({
          defaults: { from: 'Magnet <noreply@example.com>' },
        }),
      ],
      { email: { /* layout: EmailLayout */ } },
    ),
  ],
})
export class AppModule {}
```

## Environment variables

| Variable | Required | Description |
|----------|----------|-------------|
| `RESEND_API_KEY` | Yes | Resend API key |
| `EMAIL_FROM` | No | Default `from` if not passed in `defaults` |
