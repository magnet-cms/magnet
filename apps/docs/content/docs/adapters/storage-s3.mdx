# Amazon S3 storage adapter

`@magnet-cms/adapter-storage-s3` registers a `StorageMagnetProvider` for AWS S3 or **S3-compatible** endpoints (MinIO, etc.).

## Installation

```bash
bun add @magnet-cms/adapter-storage-s3 @aws-sdk/client-s3 @aws-sdk/s3-request-presigner
```

## Registration

```typescript
import { Module } from '@nestjs/common'
import { MagnetModule } from '@magnet-cms/core'
import { MongooseDatabaseAdapter } from '@magnet-cms/adapter-db-mongoose'
import { S3StorageAdapter } from '@magnet-cms/adapter-storage-s3'

@Module({
  imports: [
    MagnetModule.forRoot([
      MongooseDatabaseAdapter.forRoot(),
      S3StorageAdapter.forRoot({
        bucket: 'my-media',
        region: 'us-east-1',
        publicUrl: 'https://cdn.example.com',
      }),
    ]),
  ],
})
export class AppModule {}
```

## Environment variables

| Variable | Required | Description |
|----------|----------|-------------|
| `S3_BUCKET` | Yes* | Bucket name |
| `S3_REGION` | No | Region (default `us-east-1`) |
| `S3_ACCESS_KEY_ID` | Yes* | Access key |
| `S3_SECRET_ACCESS_KEY` | Yes* | Secret key |
| `S3_ENDPOINT` | No | Custom endpoint (MinIO, etc.) |
| `S3_PUBLIC_URL` | No | Public base URL for objects |

\*Omit if you pass everything in `forRoot()`.
