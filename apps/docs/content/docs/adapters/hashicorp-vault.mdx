# HashiCorp Vault adapter

`@magnet-cms/adapter-vault-hashicorp` registers a `VaultMagnetProvider` for HashiCorp Vault **KV version 2**.

## Installation

```bash
bun add @magnet-cms/adapter-vault-hashicorp
```

## Registration

```typescript
import { Module } from '@nestjs/common'
import { MagnetModule } from '@magnet-cms/core'
import { MongooseDatabaseAdapter } from '@magnet-cms/adapter-db-mongoose'
import { HashiCorpVaultAdapter } from '@magnet-cms/adapter-vault-hashicorp'

@Module({
  imports: [
    MagnetModule.forRoot([
      MongooseDatabaseAdapter.forRoot(),
      HashiCorpVaultAdapter.forRoot(),
    ]),
  ],
})
export class AppModule {}
```

With explicit URL and token:

```typescript
HashiCorpVaultAdapter.forRoot({
  url: 'https://vault.example.com:8200',
  mountPath: 'secret',
  auth: { type: 'token', token: process.env.VAULT_TOKEN },
})
```

AppRole and other auth modes are supported via the adapter’s `HashiCorpVaultConfig` type.

## Environment variables

| Variable | Required | Description |
|----------|----------|-------------|
| `VAULT_ADDR` | Yes* | Vault server URL |
| `VAULT_TOKEN` | No* | Token auth |

\*Required when not passed in `forRoot()` config.
