# Bem-vindo ao Magnet

Magnet é um framework CMS headless moderno construído em NestJS com suporte nativo para Bun. Ele fornece uma arquitetura flexível para construir sistemas de gerenciamento de conteúdo com uma interface administrativa poderosa.

## Recursos

- **NestJS Core**: Construído sobre o robusto framework NestJS para aplicações corporativas
- **Agnóstico de Banco de Dados**: Inclui adaptador Mongoose, facilmente extensível para outros bancos
- **Interface Admin**: Interface de administração React bonita com gerenciamento de conteúdo
- **Sistema de Plugins**: Estenda funcionalidades com plugins como Content Builder e SEO
- **TypeScript First**: Suporte completo a TypeScript em toda a stack
- **Descoberta de Schemas**: Detecção automática de schemas e extração de metadados
- **Autenticação**: Autenticação JWT integrada com guards e estratégias

## Início Rápido

```bash
# Instalar pacotes principais
bun add @magnet/core @magnet/adapter-mongoose @magnet/common
```
