# Admin Module

The Admin module provides the backend API for the admin interface.

## Admin Service

The `AdminService` provides methods for content management:

```ts
import { Injectable } from '@nestjs/common'
import { AdminService } from '@magnet-cms/core'

@Injectable()
export class MyService {
  constructor(private adminService: AdminService) {}
}
```

## API Endpoints

The admin module exposes the following REST endpoints:

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/admin/api/schemas` | List all registered schemas |
| GET | `/admin/api/schemas/:name` | Get schema metadata |
| GET | `/admin/api/content/:schema` | List content items |
| GET | `/admin/api/content/:schema/:id` | Get single item |
| POST | `/admin/api/content/:schema` | Create item |
| PUT | `/admin/api/content/:schema/:id` | Update item |
| DELETE | `/admin/api/content/:schema/:id` | Delete item |

## Configuration

```ts
MagnetModule.forRoot({
  admin: {
    enabled: true,
    basePath: '/admin',
  },
})
```

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `enabled` | boolean | `true` | Enable/disable admin module |
| `basePath` | string | `'/admin'` | Base path for admin routes |

## Accessing Admin UI

Once configured, the admin UI is available at:

```
http://localhost:3000/admin
```
