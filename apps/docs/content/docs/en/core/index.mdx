# @magnet-cms/core

The core package provides the main `MagnetModule` which orchestrates all CMS functionality including admin services, authentication, content management, and database operations.

## Installation

```bash
bun add @magnet-cms/core
```

## Module Architecture

The core module consists of several sub-modules:

| Module | Description |
|--------|-------------|
| **Admin** | Admin API and service layer for content management |
| **Auth** | Authentication and authorization with JWT |
| **Content** | Content management APIs and services |
| **Database** | Database abstraction layer for adapters |
| **Discovery** | Schema and route discovery services |

## Basic Usage

```ts
import { Module } from '@nestjs/common'
import { MagnetModule } from '@magnet-cms/core'
import { MongooseAdapter } from '@magnet-cms/adapter-mongoose'

@Module({
  imports: [
    MagnetModule.forRoot({
      adapter: new MongooseAdapter({
        uri: process.env.MONGODB_URI,
      }),
    }),
  ],
})
export class AppModule {}
```
