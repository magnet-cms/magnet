# SEO Plugin

The SEO plugin adds search engine optimization features to your content.

## Installation

```bash
bun add @magnet/plugin-seo
```

## Configuration

```ts
import { SeoPlugin } from '@magnet/plugin-seo'

MagnetModule.forRoot({
  plugins: [
    new SeoPlugin({
      defaultTitle: 'My Website',
      titleTemplate: '%s | My Website',
      defaultDescription: 'Default meta description',
    }),
  ],
})
```

### Options

| Option | Type | Description |
|--------|------|-------------|
| `defaultTitle` | string | Default page title |
| `titleTemplate` | string | Title template pattern |
| `defaultDescription` | string | Default meta description |

## Schema Integration

Add SEO fields to your schemas:

```ts
import { Schema, Prop, UI } from '@magnet/common'
import { Seo } from '@magnet/plugin-seo'

@Schema({ collection: 'posts' })
export class Post {
  @Prop()
  title: string

  @Prop()
  @Seo()
  seo: SeoMetadata
}
```

## SEO Fields

The plugin adds these fields:

| Field | Description |
|-------|-------------|
| `title` | SEO title (meta title) |
| `description` | Meta description |
| `keywords` | Meta keywords |
| `ogTitle` | Open Graph title |
| `ogDescription` | Open Graph description |
| `ogImage` | Open Graph image |
| `canonical` | Canonical URL |
| `noIndex` | Exclude from search engines |

## Admin UI

The plugin adds an SEO section in the admin editor with:

- Title preview
- Description character count
- Open Graph preview
- Search result preview
