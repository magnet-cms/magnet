# Content Builder Plugin

The Content Builder plugin provides a visual, block-based content editor.

## Installation

```bash
bun add @magnet/plugin-content-builder
```

## Configuration

```ts
import { ContentBuilderPlugin } from '@magnet/plugin-content-builder'

MagnetModule.forRoot({
  plugins: [
    new ContentBuilderPlugin({
      // Plugin options
    }),
  ],
})
```

## Features

- Drag-and-drop block editor
- Pre-built content blocks
- Custom block creation
- Responsive preview

## Using in Schemas

```ts
@Schema({ collection: 'pages' })
export class Page {
  @Prop()
  @UI({ component: 'content-builder' })
  content: ContentBlock[]
}
```

## Built-in Blocks

| Block | Description |
|-------|-------------|
| Text | Rich text content |
| Image | Image with caption |
| Gallery | Image gallery |
| Video | Embedded video |
| Quote | Blockquote |
| Code | Code snippet |
| Divider | Section divider |

## Custom Blocks

Create custom blocks:

```ts
import { defineBlock } from '@magnet/plugin-content-builder'

const ctaBlock = defineBlock({
  name: 'cta',
  label: 'Call to Action',
  fields: [
    { name: 'title', type: 'text' },
    { name: 'buttonText', type: 'text' },
    { name: 'buttonUrl', type: 'url' },
  ],
  render: (data) => `
    <div class="cta">
      <h2>${data.title}</h2>
      <a href="${data.buttonUrl}">${data.buttonText}</a>
    </div>
  `,
})
```
