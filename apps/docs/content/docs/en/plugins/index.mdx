# Plugins

Plugins extend Magnet functionality with additional features.

## Available Plugins

| Plugin | Package | Description |
|--------|---------|-------------|
| Content Builder | `@magnet/plugin-content-builder` | Visual content builder |
| SEO | `@magnet/plugin-seo` | SEO metadata management |

## Using Plugins

Register plugins in your MagnetModule configuration:

```ts
import { MagnetModule } from '@magnet/core'
import { ContentBuilderPlugin } from '@magnet/plugin-content-builder'
import { SeoPlugin } from '@magnet/plugin-seo'

MagnetModule.forRoot({
  plugins: [
    new ContentBuilderPlugin(),
    new SeoPlugin({
      defaultTitle: 'My Site',
    }),
  ],
})
```

## Plugin Architecture

Plugins can:
- Add new schema decorators
- Register API endpoints
- Add admin UI components
- Extend content services

## Creating Custom Plugins

```ts
import { MagnetPlugin } from '@magnet/core'

export class CustomPlugin implements MagnetPlugin {
  name = 'custom-plugin'

  onInit(magnet: MagnetService) {
    // Plugin initialization
  }

  getSchemas() {
    return [] // Additional schemas
  }

  getRoutes() {
    return [] // Additional routes
  }
}
```
