# Adapters

Adapters provide the database layer for Magnet, allowing you to use different databases with the same API.

## Available Adapters

| Adapter | Package | Database |
|---------|---------|----------|
| Mongoose | `@magnet/adapter-mongoose` | MongoDB |

## Choosing an Adapter

### Mongoose Adapter

Best for:
- MongoDB deployments
- Projects requiring MongoDB-specific features
- Existing MongoDB infrastructure

```bash
bun add @magnet/adapter-mongoose
```

## Adapter Interface

All adapters implement the `DatabaseAdapter` interface:

```ts
interface DatabaseAdapter {
  connect(): Promise<void>
  disconnect(): Promise<void>
  getModel(name: string): any
  createModel(name: string, schema: any): any
  findOne(model: string, id: string): Promise<any>
  findAll(model: string, query?: QueryOptions): Promise<any[]>
  create(model: string, data: any): Promise<any>
  update(model: string, id: string, data: any): Promise<any>
  delete(model: string, id: string): Promise<boolean>
}
```

## Creating Custom Adapters

You can create adapters for other databases:

```ts
import { DatabaseAdapter } from '@magnet/core'

export class PostgresAdapter implements DatabaseAdapter {
  // Implement interface methods
}
```
