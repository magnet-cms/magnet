# UI Components

The `@magnet-cms/ui` package provides reusable React components built with Radix UI and Tailwind CSS.

## Installation

```bash
bun add @magnet-cms/ui
```

## Components

### Form Components

#### Input

```tsx
import { Input } from '@magnet-cms/ui'

<Input
  label="Email"
  placeholder="Enter email"
  type="email"
  error="Invalid email"
/>
```

#### Select

```tsx
import { Select } from '@magnet-cms/ui'

<Select
  label="Category"
  options={[
    { value: 'tech', label: 'Technology' },
    { value: 'news', label: 'News' },
  ]}
/>
```

#### Switch

```tsx
import { Switch } from '@magnet-cms/ui'

<Switch
  label="Published"
  checked={published}
  onCheckedChange={setPublished}
/>
```

### Layout Components

#### Card

```tsx
import { Card, CardHeader, CardContent } from '@magnet-cms/ui'

<Card>
  <CardHeader>Title</CardHeader>
  <CardContent>Content here</CardContent>
</Card>
```

#### Dialog

```tsx
import { Dialog, DialogTrigger, DialogContent } from '@magnet-cms/ui'

<Dialog>
  <DialogTrigger>Open</DialogTrigger>
  <DialogContent>
    Dialog content
  </DialogContent>
</Dialog>
```

### Data Display

#### Table

```tsx
import { DataTable } from '@magnet-cms/ui'

<DataTable
  columns={columns}
  data={data}
  pagination
  sorting
/>
```

#### Badge

```tsx
import { Badge } from '@magnet-cms/ui'

<Badge variant="success">Published</Badge>
<Badge variant="warning">Draft</Badge>
```

## Theming

Components support theming via CSS variables:

```css
:root {
  --primary: 220 90% 56%;
  --primary-foreground: 0 0% 100%;
}
```

## Tailwind Integration

The package is designed to work with Tailwind CSS v4:

```css
@import '@magnet-cms/ui/styles.css';
```
