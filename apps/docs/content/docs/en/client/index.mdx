# Client Packages

Magnet provides React-based frontend packages for building admin interfaces and UI components.

## Packages

| Package | Description |
|---------|-------------|
| `@magnet-cms/admin` | Complete admin interface |
| `@magnet-cms/ui` | Shared UI component library |

## @magnet-cms/admin

The admin package provides a complete content management interface:

```bash
bun add @magnet-cms/admin
```

Features:
- Content manager with CRUD operations
- Schema-driven forms
- Authentication integration
- Dark/light theme support

## @magnet-cms/ui

The UI package provides reusable React components:

```bash
bun add @magnet-cms/ui
```

Features:
- Form components (Input, Select, Switch, etc.)
- Layout components
- Data display components
- Radix UI based primitives
