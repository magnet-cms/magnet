# Welcome to Magnet

Magnet is a headless CMS framework built on NestJS: schema-driven content, a React admin UI, and a **provider-based** bootstrap so you can mix databases, auth, storage, email, cache, vault, GraphQL, and plugins in one `MagnetModule.forRoot([...])` call.

## Features

- **NestJS-native** — Modules, DI, and validation patterns you already use
- **Headless API** — REST (and optional GraphQL) for any frontend
- **Composable providers** — One array for DB + Supabase/Clerk auth + S3/R2 storage + Resend email + Redis cache + Vault + plugins; required env vars validated at startup
- **Admin UI** — Served from the core app or embedded (e.g. Next.js) via `@magnet-cms/admin`
- **Drizzle migrations** — First-class migration workflow for SQL databases
- **RBAC, webhooks, notifications, settings** — Built-in modules documented under [Core](/docs/core)

## Packages (overview)

| Package | Description |
|---------|-------------|
| `@magnet-cms/core` | `MagnetModule`, HTTP modules, admin serving |
| `@magnet-cms/common` | Decorators, types, `MagnetProvider` contracts |
| `@magnet-cms/admin` | Admin UI bundle |
| `@magnet-cms/adapter-db-mongoose` | MongoDB |
| `@magnet-cms/adapter-db-drizzle` | PostgreSQL / MySQL / SQLite |

See [Adapters](/docs/adapters) for auth, storage, email, cache, vault, and GraphQL packages.
