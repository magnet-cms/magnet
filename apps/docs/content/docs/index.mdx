# Welcome to Magnet

Magnet is a headless CMS framework built on NestJS: schema-driven content, a React admin UI, and a **provider-based** bootstrap so you can mix databases, auth, storage, email, cache, vault, GraphQL, and plugins in one `MagnetModule.forRoot([...])` call.
