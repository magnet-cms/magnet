# Welcome to Magnet

Magnet is a headless CMS framework built on NestJS, designed to provide a flexible and extensible content management system for modern applications.

## Features

- **NestJS-based** - Built on top of the powerful NestJS framework
- **Headless Architecture** - API-first approach for maximum flexibility
- **Plugin System** - Extend functionality with plugins
- **Admin UI** - Built-in React admin interface
- **Database Adapters** - Support for multiple databases through adapters

## Getting Started

Ready to start building with Magnet? Check out our [Quick Start](/docs/getting-started/quick-start) guide to get up and running in minutes.

## Packages

Magnet is organized as a monorepo with several packages:

| Package | Description |
|---------|-------------|
| `@magnet/core` | Core NestJS module |
| `@magnet/common` | Shared types and utilities |
| `@magnet/admin` | React admin UI |
| `@magnet/adapter-mongoose` | Mongoose database adapter |
